Entrega 2

En esta sección se mostrará como fue el procedimiento para replicar un modelo de silla.

Pasos a seguir para poder replicar esta silla

  1. Separar y replicar cada una de las piezas lisas, fuera del bosquejo original de la silla.

  2. Preparar una “caja” dentro del programa Fusion, con tal de representar la plancha de madera de la cual salen todas las piezas de la silla. Cabe mencionar que la ilustración se realizó con una plancha de 15 mms.

  3. Posteriormente, se deben realizar los ajustes de corte. Para ello nos debemos situar en la pestaña de manufacutra, pinchar en setup, y establecer los primeros parámetros. Este “seteo” permite decirle al programa bajo que condiciones son las que tiene que trabajar, como lo que sería el espacio asignado, que tiene que moldear, bajo que medida,etc.

Se recomienda elegir la orientación a partir de la caja recientemente constuida, y el modelo a afectar serían las piezas que se encuentran dentro. Como lo sería la configuración de que el eje Z sea la cara superior de la caja y el x un borde. Y finalmente el boxpoint, que sería un punto asignado a una esquina en la cara superior de nuestra caja.

Empezamos con el mecanizado

Después de realizar todo el setup, vendría elegir la tarea que vamos a realizar

  1. Con el fin de realizar el calado que tienen las unines de las piezas, para poder facilitar el encajado, se escoge aquellas piezas que disponen de esta cualidad. Cabe destacar que el diseño curvo o redondo que estos tienen, es debido a la forma en que funciona la fresa, por eso no puede haber ángulos rectos, dentro de lo que implican los bordes.

Para ello se selecciona la herramienta “contour 2d”

Utilizando esa opción, se procede a seleccionar el borde interno de las orejas en cada una de las piezas. Cabe mencionar que se formaran unas líneas que son el contorno recientemente elegido. Adicionalmente, se abrirá una ventana de configuraciones, en donde se selecciona la fresa a utilizar y el comportamiento que tendrá el cabezal con la fresa, mientras se realiza el contour. Posteriormente, se recomienda visualizar como sería el resultado del contour, por lo tanto se procede a hacer click izquierdo en la pestaña del contour y seleccionar Simular. Hacer esa acción abre una ventana similar a lo siguiente.

Pero hay un problema, al realizarlo así se esfa forzando mucho el trabajo de la fresa, lo cual puede malograr tanto la pieza como la máquina, por lo tanto se recomienda hacer un paso extra. Esta adición permite hacer un trabajo progresivo y que no forzar que se haga la pieza. Como dato, el ajuste que se debe realizar, debiese corresponder a la mitaddel tamaño de la fresa. No obstante, el tamaño puede ser más pequeño y entrega un mejor acabado. 5) Para realizar el calado que tienen las piezas, se utiliza la herramienta pocket. Lo cual nos entrega una ventana similar a esta Nos aparecerá un cuadro donde se tendrá que elegir la fresa y como se pretende mover el cabeza, de igual manera que en el contour. Posteriormente, realizaremos la simulación.
6) Como paso final, Se realiza un contour 2d al contorno de las piezas, de igual manera que se realizó anteriormente. Cabe destacar que, al realizar esta tarea, la pieza se saldrá del material, por lo tanto se recomienda realizar unos tabs(que se encuentran dentro de la misma ventana del contour), con el fin de que no ocurra ningún accidente. Estas piezas se pueden hacer de esa forma debido a que corresponden a piezas lisas o derechas, sin ninguna curvatura. Si la pieza tuviese la característica de ser curva, como el asiento, se tiene que realizar el siguiente procedimiento.

Ahora el asiento

  1. Para realizar el asiento, se debe tener en cuenta la cantidad de material necesario para realizarlo. Por lo tanto, se crea una rectángulo debajo del modelo y se le aplica un extrude, para generar un cuerpo, de 9mm. 2)Ahora tenemos que volver a la sección manufactura y armar el setup, de igual manera que se realizó anteriormente. La diferencia, es que ahora usaremos la herramienta Trace 3)Teniendo seleccionado el objeto, se procede a seleccionar cada una de las lineas que tiene el asiento en sí. Adicionalmente, se debe hacer una configuración en “Stock to leave”, con el fin de que se haga un calado fino hasta un límite, permitiendo que el material se doble y facilite el asiento.
  2. Finalmente, como siempre, se recomienda realizar una simulación para ver como quedaría el asiento.
